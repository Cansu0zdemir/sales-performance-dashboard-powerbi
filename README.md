# Sales Performance Dashboard (Power BI) | AdventureWorks

This project showcases an **Executive Sales Performance Dashboard** built in **Power BI** using the AdventureWorks dataset and a star schema model.

## Pages

### 1) Executive Summary
- KPI Cards: **Revenue**, **Profit**, **Margin %**, **YoY Growth %**
- Monthly trend: **Revenue vs Revenue LY**
- **Revenue by Product Category**
- Conditional formatting for YoY Growth (negative = red, positive = green)

### 2) Sales Analysis
- **Revenue by Country** (Map)
- **Top 10 Products by Revenue** (Top N filter + descending sort)

## Key Skills Demonstrated
- Data modeling with a **star schema** (fact/dimension structure)
- DAX measures (KPIs + time intelligence)
- Interactive visuals, slicers, and cross-filtering
- Business-focused layout and clean reporting design

## How to Use
1. Download the `.pbix` file from this repository
2. Open it with **Power BI Desktop**
3. Use the **Year** slicer to explore performance over time
